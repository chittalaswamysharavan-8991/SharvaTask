# V2.1 Widget Upgrade

## Goal

Make SharvaTask feel closer to 1-2-3 List by rendering an interactive list widget inside ChatGPT, while keeping V2's Vercel + Blob persistent history backend.

## Completed

1. Current V2 backend kept as-is.
2. Added widget HTML component.
3. Attached widget to tools using:
   - `_meta.ui.resourceUri`
   - `_meta["openai/outputTemplate"]`
4. Tool responses now include `structuredContent`.
5. Widget buttons added:
   - Add task
   - Done/reopen
   - Block
   - Archive
   - Refresh
   - History
6. Build passed with `npm run build`.

## Tool output shape

```ts
{
  view: 'list' | 'lists' | 'history' | 'message',
  message: string,
  list?: SharvaTaskList,
  lists?: ListSummary[],
  events?: SharvaTaskEvent[],
  query?: string
}
```

## Main success test

Ask ChatGPT:

```txt
Show my lists using SharvaTask MCP.
```

Expected:

- An interactive widget/card appears.
- List cards have Open/Continue buttons.
- Opening a list shows task rows.
- Add/done/archive actions update through MCP tools.
