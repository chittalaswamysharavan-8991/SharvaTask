# SharvaTask MCP V2.3 — Glass Lite

This patch refines the V2.2 milky-white widget into a lighter, more Apple-glass inspired interface.

## Goals
- Lighter visual weight
- Softer translucency
- Reduced boxy feel
- Cleaner stat summaries
- Softer list cards and task rows

## Main changes
- Main widget card uses translucent glass background with blur.
- Buttons, task rows, list cards, and empty states use softer borders and more floating surfaces.
- Stats simplified into lighter pill-like cards.
- Composer and task sections use lower contrast containers for a more airy feel.

## Deploy
```bash
npm install
npm run build
npx vercel --prod
```
