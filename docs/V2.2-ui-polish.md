# SharvaTask MCP V2.2 — UI Polish Pass

## Scope

V2.2 is a UI/UX polish-only release. It keeps the V2/V2.1 backend, Vercel deployment model, Vercel Blob event storage, and MCP tool contracts intact.

No backend redesign, no storage migration, no paid service, and no new cloud dependency were added.

## Completed UI improvements

- Milky white widget theme for better visibility on ChatGPT dark mode.
- Stronger card shell with warm off-white background, soft border, rounded corners, and deeper but subtle shadow.
- Clearer header hierarchy: small product label, strong list title, project/status/updated metadata, and count summary cards.
- Cleaner task composer with taller input, priority dropdown, Add CTA, Enter-to-add, and disabled Add when empty.
- Task rows redesigned for readability: bigger task title, visible checkbox, status/priority metadata, updated timestamp, and hidden task ID under Details.
- Sectioned task layout: Pending, Blocked, and Done sections with counts. Large Done sections collapse automatically.
- Cleaner Your Lists grid with milky cards, list stats, last updated time, and Open/Continue actions.
- Better empty states for no tasks and no lists.
- Better loading/success/error feedback via toast state.
- Responsive improvements for narrow/mobile widths.
- Shorter MCP textual result output so the widget carries the UI and duplicate text/table clutter is reduced.

## Deploy

```bash
npm install
npm run build
npx vercel --prod
```

After deploy, health should show:

```json
"version": "2.2.0-ui-polish"
```

Then reload/reconnect the ChatGPT connector and test:

```txt
Show my lists using SharvaTask MCP.
```

Expected: a milky-white widget/card UI renders with cleaner list visibility.
